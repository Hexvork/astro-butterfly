I originally used the Stellar blog theme, then switched to the Butterfly theme on a friend's recommendation, but got stuck on deploying the friends link page. After many trials and errors, I finally successfully built a satisfactory friends link page, so I sorted out this step-by-step tutorial hoping to help other beginners.

### I. Create a Page File
Create a file named `index.md` in `blog/source/link`
Open the file and modify the frontmatter as follows:
```markdown
---
title: Friends Links
date: 2025-11-16
type: "link"
---
```

### II. Create a Data File
Create a file named `link.yml` in `blog/_data`
Open the file and modify it as follows:
```yml
- class_name: Friends Links
  class_desc: Every site here is intriguing...
  flink_style: flexcard
  link_list:
    - name: Yuyan Blog
      link: https://www.yuyano.com
      avatar: https://www.yuyano.com/img/0/favicon.webp
      descr: A technical blog
```

#### Field Explanations:
- `name`: Website name
- `link`: Website URL
- `avatar`: Website avatar/icon link (corrected from original typo: not "website email")
- `descr`: Website description

To add more friends links, simply add additional groups of `name`, `link`, `avatar`, and `descr` fields.

### III. Display Friends Link in Navigation Bar
Open `_config.yml` in `blog/themes/butterfly`
Add the following line to the `menu` section:
```yml
Friends Links: /link/ || fas fa-link
```

Finally, execute the Hexo three-step deployment commands:
```txt
hexo cl
hexo g
hexo d
```

If everything goes well, the friends link page should display normally.

## Summary
Building a website is essentially an exploratory DIY journey. Every problem solved after encountering pitfalls is your own progress. Go ahead and refresh your blog to check out your exclusive friends link display area!
