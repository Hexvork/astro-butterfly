## ⚠ Important: Replace CDN

#### Single Resource Replacement

Example: [https://cdn.jsdelivr.net/npm/jquery@3.6.4/dist/jquery.min.js](https://cdn.jsdelivr.net/npm/jquery@3.6.4/dist/jquery.min.js) --> [https://cdn.jsdmirror.com/npm/jquery@3.6.4/dist/jquery.min.js](https://cdn.jsdmirror.com/npm/jquery@3.6.4/dist/jquery.min.js) 

#### Overall Replacement for Butterfly Theme

Open `cdn.js` in `blog\themes\butterfly\scripts\events`

Replace the content with the following (taking JSDMirror as an example) 

```js
      const cdnSource = {
        local: cond === 'internal' ? `${cdnjs_file + verType}` : `/pluginsSrc/${name}/${file + verType}`,
        jsdelivr: `https://cdn.jsdelivr.net/npm/${name}${verType}/${min_file}`,
        unpkg: `https://unpkg.com/${name}${verType}/${file}`,
        cdnjs: `https://cdnjs.cloudflare.com/ajax/libs/${cdnjs_name}/${version}/${min_cdnjs_file}`,
        jsdmirror : `https://cdn.jsdmirror.com/npm/${name}${verType}/${min_file}`,
      }
```

Then, in `blog\themes\butterfly`, open `_config.yml`, find the line for CDN selection, and modify it as follows:

```yml
CDN:
  internal_provider: local
  third_party_provider: jsdmirror
```

If everything goes well, your CDN has been successfully replaced.
