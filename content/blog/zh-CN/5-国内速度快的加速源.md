## ⚠重点 替换CDN

#### 单个资源替换

例: [https://cdn.jsdelivr.net/npm/jquery@3.6.4/dist/jquery.min.js](https://cdn.jsdelivr.net/npm/jquery@3.6.4/dist/jquery.min.js) --> [https://cdn.jsdmirror.com/npm/jquery@3.6.4/dist/jquery.min.js](https://cdn.jsdmirror.com/npm/jquery@3.6.4/dist/jquery.min.js) 

#### butterfly主题整体替换

打开`blog\themes\butterfly\scripts\events`中的`cdn.js`

把这段内容替换成这样(我以JSDMirror为例) 

```js
      const cdnSource = {
        local: cond === 'internal' ? `${cdnjs_file + verType}` : `/pluginsSrc/${name}/${file + verType}`,
        jsdelivr: `https://cdn.jsdelivr.net/npm/${name}${verType}/${min_file}`,
        unpkg: `https://unpkg.com/${name}${verType}/${file}`,
        cdnjs: `https://cdnjs.cloudflare.com/ajax/libs/${cdnjs_name}/${version}/${min_cdnjs_file}`,
        jsdmirror : `https://cdn.jsdmirror.com/npm/${name}${verType}/${min_file}`,
      }
```

然后，在`blog\themes\butterfly`中，打开`_config.yml` 找到CDN选择那里行，改成这样：

```yml
CDN:
  internal_provider: local
  third_party_provider: jsdmirror
```

不出意外，你的CDN已经成功替换了
